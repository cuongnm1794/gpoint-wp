== Description ==

BusinessOrange is a modern WordPress block theme inspired by ThemeWagon Business template. It features a beautiful orange color scheme, side menu navigation, tabs functionality, and comprehensive sections for business websites.

== Features ==

* Full Site Editing (FSE) support
* Block-based theme architecture
* Responsive design (mobile, tablet, desktop)
* Custom color palette with orange as primary color
* Side menu navigation
* About section with tabs (Who We Are, Vision, History)
* Services section with 6 services
* Pricing section with 3 plans
* CTA section
* Blog/Latest News section
* Clients logo carousel (using Swiper.js)
* Contact form section
* Custom fields for easy content management
* Smooth animations and transitions
* Mobile-friendly navigation

== Installation ==

1. Upload the theme folder to `/wp-content/themes/`
2. Activate the theme through the 'Appearance' menu in WordPress
3. Go to Pages > Add New to create your homepage
4. Use the "Home" template for the homepage
5. Configure custom fields in the page editor

== Customization ==

The theme includes custom meta boxes for:
* Hero Section (title, description, buttons)
* About Section (tabs content)
* Services (6 services with icons)
* Pricing Plans (3 plans)
* Clients Logos
* Contact Information

All content can be managed from the WordPress admin panel.
