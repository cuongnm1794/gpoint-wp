== Description ==

GPoint is a modern WordPress block theme inspired by GapOne design. It features a beautiful orange color scheme, fully responsive design, and comprehensive custom fields for easy content management.

== Features ==

* Full Site Editing (FSE) support
* Block-based theme architecture
* Responsive design (mobile, tablet, desktop)
* Custom color palette with orange as primary color
* Hero slider with autoplay
* Stats section with 6 statistics
* Solutions grid with 4 main solutions
* Industries section
* Why Choose Us section with 6 items
* Partners/Logos section
* Contact form section
* Custom fields for easy content management
* Smooth animations and transitions
* Mobile-friendly navigation

== Installation ==

1. Upload the theme folder to `/wp-content/themes/`
2. Activate the theme through the 'Appearance' menu in WordPress
3. Go to Pages > Add New to create your homepage
4. Use the "Home" template for the homepage
5. Configure custom fields in the page editor

== Customization ==

The theme includes custom meta boxes for:
* Hero Slider (title, description, image, CTA)
* Stats Section (6 statistics)
* Solutions (4 solutions with details)
* Why Choose Us (6 items)
* Partners Logos
* Contact Information

All content can be managed from the WordPress admin panel.
